Add Unit Master Entry
1) Click the “AddNew” button.Then you get the following screen:
2) Enter the Unit.Eg :You can use "Gram per Piece" as Unit.
3) You can set Unit for Grade, Count or Both , you should select one of these from the drop-down list named as Recieved By.
Note: In Process Code, if you select "Recieved By " as Grade then you will get all uints set for Grade only and Both.If you select Count you will get all units assigned for Count only and Both.
4) Click the Add button.You will be prompted for a confirmation.Click on the Ok button.The newly added entry will be visible on the Unit Master list
Edit Unit Master Entry
1)Click the "Edit" button .Then you get the following screen:
2) You can edit the Unit and/or Recieved By.
3) Click on the "Save Changes" button.You will be prompted for confirmation.Click on Ok button.The changes will be visible in the Unit Master list.
Delete Unit Master Entry
1) You can select more than one entries by selecting the checkboxes for deletion.
2) Click the Delete button.You will be prompted for confirmation.(as shown below):
Click on the Ok buton.The selected entry(entries) will be removed from the Unit Master list.
