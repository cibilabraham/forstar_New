View Process Code
1) By default you get entire list of the process code entries as shown in the figure.
2) You can view the process code entries of the selected fish(First select a fish from the drop down list as shown in the figure.Then you will get the process code of the selected fish).
Add Process Code Entry
1) Click the “AddNew” button.Then you get the following screen:
2) Select a fish from the drop-down list.
3) Enter the process code. Eg: You can use HL for Head Less.
4) Process description is optional.
5)There are two options to set the data.
Option 1:   You could copy all settings of an existing fish/Process Code.
  a) Select a fish from the drop-down list.
  b) Select the Process code.
        OR
Option 2:  Follow the remaining steps.
6)Enter the basket weight.
7)For raw material,you should select Count or Grade from the drop-down list named as "Recieved By".
8)If you selected the Grade,you will get all grades.
9)You can select multiple grades from the drop-down list.
10)Select the Unit.
11)For frozen material,you should select Count or Grade from the drop-down list named as "Recieved By".
12)By default the selection is Grade.In this case you can select multiple grades from the drop-down list.
13)Select the Unit.
14) Click the Add button.You will be prompted for a confirmation.Click on the Ok button.The newly added entry will be visible in the ProcessCode list .
Edit Process Code Entry
1)Click the "Edit" button .Then you get the following screen:
2) You can edit the fish name,process code,process description,basket weight,Raw material(Receivec By,Grade,Unit) and/or Frozen material(Received By,Grade,Unit).
3) Click on the "Save Changes" button.You will be prompted for confirmation.Click on the Ok button.The changes will be visible in the ProcessCode list .
Delete Process Code Entry
1) You can select more than one entries by selecting the checkboxes against each entry.
2) Click the Delete button.You will be prompted for confirmation.(as shown below):
Click on the Ok buton.The selected entry(entries) will be removed from the ProcessCode list .
