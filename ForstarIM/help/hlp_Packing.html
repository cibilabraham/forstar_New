Add Packing Entry
1) Click the “AddNew” button.Then you get the following screen:
2) Enter the appropriate packing code.
3)The description field is optional.
4)Enter the packed weight(You can use kg/Lb).
5) Click the Add button.You will be prompted for a confirmation.Click on the Ok button.The newly added entry will be visible in the Packing list.
Edit Packing Entry
1)Click the "Edit" button .Then you get the following screen:
2) You can edit the packing code,description and/or packed weight.
3) Click on the "Save Changes" button.You will be prompted for confirmation.Click on the Ok button.The changes will be visible in the Packing list.
Delete Packing Entry
1) You can select more than one entries by selecting the checkboxes against each entry.
2) Click the Delete button.You will be prompted for confirmation.(as shown below):
Click on the Ok buton.The selected entry(entries) will be removed from the Packing list.
